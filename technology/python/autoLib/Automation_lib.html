# 读取Excel文件

读取Excel文件的Python库有很多，以下是一些常用的：

  1. xlrd：xlrd是一个用于读取Excel文件的库，支持.xls格式的文件。它可以读取Excel文件的内容效率高，但不支持写入。
  2. pandas：pandas是一个流行的数据分析库，可以方便地读取、处理和分析Excel文件。它支持多种文件格式，包括.xls和.xlsx。
  3. openpyxl：openpyxl是一个用于处理Excel 2010 xlsx/xlsm/xltx/xltm文件的库。它可以读取和写入Excel文件，支持公式、图表等功能。
  4. xlwt：xlwt是一个用于写入Excel文件的库，支持.xls格式的文件。它可以创建和编辑Excel文件效率高，但不支持读取。
  5. pandas-xlrd：pandas-xlrd是一个扩展库，结合了pandas和xlrd的功能，可以直接读取Excel文件到pandas DataFrame中。
  6. pyexcel：pyexcel是一个通用的表格数据读取库，可以读取多种文件格式，包括Excel文件。它支持多种文件格式，包括.xls和.xlsx。
  7. xlsxwriter：xlsxwriter是一个用于创建Excel文件的库，支持.xlsx格式的文件。它可以创建和编辑Excel文件，但不支持读取。

这些库都有各自的优缺点，可以根据具体需求选择适合自己的库来读取Excel文件。

关于读取Excel的优质博客：https://blog.csdn.net/lm_love_hxl/article/details/130733070

# .xls和.xlsx

Excel会有.xls和.xlsx文件的原因是：二者是不同版本的Excel生成的默认文件格式1。

.xls是Excel 97-2003版本的默认文件格式，

而.xlsx是Excel 2007及更高版本的默认文件格式

.xls和.xlsx文件的区别有：

  1. 文件结构不同：.xls文件是以二进制格式存储的，不易被其他程序识别和处理；.xlsx文件则是基于开放文档标准(Open XML)的XML文件格式，数据以分层结构存储，可被其他软件解读和编辑。
  2. 文件大小不同：.xls文件以二进制格式存储，文件体积相对较大；.xlsx文件采用了更高效的压缩算法，在相同数据量的情况下，文件大小通常比.xls格式小得多。
  3. 兼容性与互操作性不同：.xls格式的文件在不同电子表格应用程序之间的互操作性较差，而.xlsx格式则相对更易于在不同软件之间共享和传输。

在Python中，你可以使用xlrd，pandas和openpyxl等库来读取Excel文件。

# xlrd

xlrd是一个用于读取Excel文件的库。它可以读取Excel文件的内容，但不支持写入。

使用xlrd库读取Excel文件的步骤如下：

import xlrd

# 读取.xls文件
data_xls = xlrd.open_workbook('file.xls')
sheet = data_xls.sheet_by_index(0)
for row in range(sheet.nrows):
    print(sheet.row_values(row))

# 读取.xlsx文件
data_xlsx = xlrd.open_workbook('file.xlsx')
sheet = data_xlsx.sheet_by_index(0)
for row in range(sheet.nrows):
    print(sheet.row_values(row))

python第三方库xlrd不支持读取.xlsx格式的Excel文件的问题详解

python第三方库xlrd读取.xlsx格式的Excel文件时报错，报错信息如下：

报错原因：

①查看本机安装的python第三方库xlrd的版本：

可以看到当前xlrd库的版本号为2.0.1；且xlrd 2.0.1版本仅支持读取.xls 格式的Excel文件。不支持读取.xlsx格式的Excel文件。

解决办法：

①如果不想降低第三方库xlrd（2.0.1 ）的版本，可以直接修改Excel文件的格式为.xls格式即可。

②降低第三方库xlrd的版本至1.2.0。

  • 卸载xlrd 2.0.1
  • 安装1.2.0

使用 xlrd.open_workbook 打开已存在的Excel文件时，如果添加参数 formatting_info=True 则会读取.xlsx格式的Excel文件报错。

formatting_info 参数默认为False，且当该参数为False时xlrd读取数据可以节约内存；（Excel文件底部空行，空单元格都是存在格式信息但是没有数据，该参数为False时直接会被当成空对待，即这将会裁剪任何底部、右边的空的表格）

formatting_info 参数为True时，xlrd读取Excel数据时会读取Excel文件中各种格式的信息。

使用 xlrd.open_workbook 打开已存在的Excel文件时，不添加参数 formatting_info （默认值为False）或者添加 formatting_info=False 参数，读取.xlsx格式的Excel文件数据成功

1.2.0版本的xlrd除了可以读取.xlsx格式的Excel文件，依然还可以读取.xls格式的Excel文件。

2、样式优化:

1、设置行列宽度 1）设置列的宽度 在表格之中行是按照数字来计算，列是按照英文字母顺序来计算的。但是在xlwt库中则是将它们都改成从0开始，类似于索引的对象获取方式。调用sheet.col(0)就可以获取到表格列对象，它表示的就是列A。然后设置width属性值即可修改列宽，xlwt中默认字体0的1/256为衡量单位。xlwt创建时使用的默认宽度为2960，既11个字符0的宽度。

所以我们在设置列宽时可以用如下方法：width = 256 * 20 256为衡量单位，20表示20个字符宽度。

示例代码：

for i in range (len(head)):
	sheet.col(i).width = 256*20

3、设置边框

xlwt要设置边框，需要使用Borders来设置，可以设置实线、虚线、没有边框三种，并且可以通过colour属性设置颜色。

  • DASHED：虚线
  • THIN：实线
  • NO_LINE：没有边框

示例：

style = xlwt.XFStyle()  # 创建一个样式对象，初始化样式
borders = xlwt.Borders()
#设置边框属性实线、黑色
borders.left = xlwt.Borders.THIN
borders.top = xlwt.Borders.THIN
borders.right = xlwt.Borders.THIN
borders.bottom = xlwt.Borders.THIN
borders.left_colour = 0xff
borders.right_colour = 0xff
borders.bottom_colour = 0xff
borders.top_colour = 0xff

style.borders = borders
for i in range(len(data)):
    for j in range(len(data[i])):
		sheet.write(i+1, j, data[i][j], style)
